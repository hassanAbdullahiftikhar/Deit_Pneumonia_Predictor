Model Information

Technical specifications and performance metrics.

Architecture: DeiT-Base

The Data-efficient Image Transformer (DeiT) is a vision transformer model that leverages distillation to achieve high performance on smaller datasets. Unlike traditional CNNs, DeiT uses self-attention mechanisms to capture global dependencies across the entire X-ray image.

Dataset: Kermany et al.

Trained on 5,856 validated Chest X-Rays (1,583 Normal, 4,273 Pneumonia) from pediatric patients. The model achieved competitive F1-scores during internal validation rounds.

System Workflow

Step-by-step breakdown of the AI analysis pipeline.

1. Image Preprocessing

Images are resized to 224x224, center-cropped, and normalized using the ImageNet mean and standard deviation to match the model's training distribution.

2. Transformer Inference

The image is split into patches and passed through multiple transformer layers. Self-attention weights determine which features (e.g., opacities, consolidations) are critical.

3. Explainability (Grad-CAM)

Gradients of the target class are backpropagated to the last transformer layer to create a coarse localization map of relevant regions.
